Special Characters

Article 1

Special characters and accented letters require a special set of code in order to appear correctly on the html pages. These are character entities, which you insert into your HTML code so that the browser will interpret these and show the corresponding symbols or characters you want. Examples of entities that require these codes, are apostrophes or double quotes for example

Article 2

Nice Entity is a great resource to find any code entity you need to make your HTML code correct. There is a list of quick references and most common entities that you can use. Examples of these are an ampersand, less than sign, non-breaking space, cent, registered trademark, just to name a few. Both of these websites are good places to go into order to find the right code to use in your HTML.
