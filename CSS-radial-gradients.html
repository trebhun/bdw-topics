CSS Radial Gradients

Article 1

The CSS function to create a radial gradient is radial-gradient(), and this allows you to create a visual gradient of colors radiating from the center. The radial gradient uses the center, ending shape (either circle or ellipse) and color stops. This is not a CSS color, but an image, with not intrinsic dimensions or no preferred size or ratio. The size will match the element it is applied to. If you want a repeating gradient you must use CSS repeating-radial-gradient

The syntax for radial gradients looks like this, radial-gradient(definition of the contour [ex: circle], size [length, percentage], position of the ending shape [at position]),(list of color stops [color, percentage, length)
